Jailbreak

I've never liked jailbreak. The only purpose to install it for me is ability to dualboot back to iOS 6

You may have different opinion, so I give few recommendations:
  • Remove code signing related boot-args. I'm not sure about p0sixspwn or evasi0n, but Pangu 7.1.x untether failed when I tried to boot with these boot arguments

  • Downgrade iTunes. MobileDevice framework, which is used by jailbreak utilities, updates together with iTunes. That means, older jailbreaks won't work with modern iTunes versions

  • Jailbreak process usually implies few reboots while it. Disconnect your device shortly after it starts to reboot and connect back only when second OS is about to show SpringBoard
